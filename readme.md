## Locally Run

### Type in your terminal the following

1. Install Node Modules

   - Location: `dir/final_project` && `dir/final_project/react_frontend`

   ```zsh
   npm i
   ```

2. Run the Backend and Frontend

   - Location: `dir/final_project`
   - Frontend: React
   - Backend: ASP.Net
   - Database: SQLite

   ```zsh
   npm run stack
   ```
